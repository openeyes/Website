Graphing

Description:

This page shows the graphing package developed for OpenEyes. There are many situations where it is useful to have a graphical display of one or more parameters related to the patient over time, for example IOP or visual acuity. The display should also have the ability to include significant events, such as changes in drug treatment, or operations.

The graph below shows an example. Hovering over a point can give additional information (try hovering over the January or June data points for the left eye). In a real application, clicking on a point would navigate to the event containing the data.
