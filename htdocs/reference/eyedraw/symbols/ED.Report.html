Class Detail
ED.Report(Drawing)
An object of the Report class is used to extract data for the Royal College of Ophthalmologists retinal detachment dataset. The object analyses an EyeDraw drawing, and sets the value of HTML elements on the page accordingly.
Parameters:
Drawing
_drawing The drawing object to be analysed
